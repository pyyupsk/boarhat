BUILD

WEAPON

Withershade

Skill Range Boost, which is good since Skill Range will be converted into Skill DMG upon using ULT. Since ULT will remove sanity cost of Combat Skill and boost Effect Trigger Probability, her ULT is going to be spammed.

Embla Inflorescence

Skill Range boost and Skill Damage boost which will get converted into pure Skill DMG on Lady Nifle's ULT for bossing. The weapon itself release multiple arrow at group of mobs, allowing Lady Nifle to pretty much guarantee proc her I6 on every cast.

Skill Range (+30% / 36% / 42% / 48% / 54% / 60%). While charging, gradually locks onto nearby enemies. Upon release, fires tracking arrows at all locked targets. When a Lumino character deals Skill Damage, grants (+35% / 42% / 49% / 56% / 63% / 70%) Skill DMG for 6s.
